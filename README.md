
# Employee Payroll JS

This is a simple Node.js based Employee Payroll Management project.
It demonstrates basic CRUD (Create, Read, Update, Delete) operations using JavaScript classes and in-memory storage.

The project follows a clean folder structure separating:

* Model (Employee structure)
* Service (Business logic)
* Data (In-memory storage)
* Main application file

---


##  Features

* Add new employee
* View all employees
* Find employee by ID
* Update employee details
* Delete employee
* Basic validation and error handling

---

## Technologies Used

* Node.js
* JavaScript (ES6 Classes)
* Console-based output

---

## Note

* This project uses **in-memory storage**, so data will reset every time you run the program.
* It is built for learning and practice purposes.
